# Publicar formulário

Uma vez que o formulário esteja finalizado e pronto para uso, basta publicá-lo através do botão "Publicar", tanto na listagem de seções como na listagem de versões.

<figure><img src="../../.gitbook/assets/image (200).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (221).png" alt=""><figcaption></figcaption></figure>

Ao publicar uma versão de formulário, esta ficará disponível para uso no SIAD Forms e no SIAD Benefícios.

{% hint style="info" %}
O formulário por si só, mesmo publicado, não irá funcionar se não tiver outra funcionalidade que faça uso deste.
{% endhint %}

{% hint style="warning" %}
Ao publicar versões novas de formulários que já estavam sendo utilizados anteriormente, é necessário atualizar manualmente suas respectivas versões para a mais recente.
{% endhint %}

{% hint style="danger" %}
Ao publicar uma versão de formulário, qualquer outra versão publicada passará para o status "Arquivada", não podendo mais ser utilizada.
{% endhint %}
