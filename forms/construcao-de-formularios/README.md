# Construção de Formulários

## Visão geral do processo

A construção de um formulário no SIAD Forms segue três etapas sequenciais:

1. **Criar o formulário** (nome e descrição)
2. **Criar uma versão** (que conterá toda a estrutura)
3. **Organizar seções e atributos** (campos do formulário)
4. **Publicar a versão** (tornando-a disponível para uso em fluxos)

```
Criar Formulário 
    → Criar Versão (Rascunho) 
        → Criar Seções 
            → Adicionar Atributos 
                → Publicar Versão
```
