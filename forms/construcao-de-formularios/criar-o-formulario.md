# Criar o formulário

O formulário é a entidade base. Funciona como um "contêiner" que agrupa suas versões. Neste momento, você define apenas informações identificadoras.

#### Como acessar

1. Acesso o módulo de Formulários (pelo ícone superior de seleção de módulos ou pelo link [https://siad.prefeitura.sp.gov.br/formularios](https://siad.prefeitura.sp.gov.br/formularios)).
2. No menu lateral, acesse **SIAD Forms > Formulários**
3. Clique no botão **Criar Formulário**

Útil para documentação interna.</td></tr></tbody></table>

<figure><img src="../../.gitbook/assets/image (221).png" alt=""><figcaption></figcaption></figure>

#### Após criar

O formulário será exibido na listagem. Ao clicar nele, você terá acesso à sidebar de navegação com as opções:

* Visualizar Formulário
* Editar Formulário
* Versões
